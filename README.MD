# Pygame - Colisão entre Objetos

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python)
![Pygame](https://img.shields.io/badge/Pygame-2.x-green)

## Sobre o Projeto

Este projeto foi desenvolvido utilizando Python e a biblioteca Pygame, com o objetivo de simular dois objetos se movimentando pela tela, quicando nas bordas e colidindo entre si.

A movimentação é baseada na lógica clássica do efeito "DVD".

---

## Funcionalidades

- Movimento automático dos objetos
- Colisão com as bordas da janela
- Detecção de colisão entre objetos usando `Rect.colliderect()`
- Mudança de direção ao colidir
